# 深拷贝和浅拷贝
<a name="KUmpE"></a>
## 浅拷贝
拷贝第一层引用类型
```javascript
// 浅拷贝
function shallowClone(target) {
  // 如果不是对象直接返回
  if (!target || typeof target !== 'object') return target
  const ans = target instanceof Array ? [] : {}
  Object.assign(ans, target)
  // for (let key in target) {
  //   if (target.hasOwnProperty(key)) {
  //     ans[key] = target && typeof target === 'object' ? shallowClone(target[key]) : target[key]
  //   }
  // }
  return ans
}
```

## 深拷贝
拷贝所有的引用类型
```javascript
function deepClone(target) {
  if (!target || typeof target !== 'object') return target
  const ans = target instanceof Array ? [] : {}
  // Object.assign(ans, target)
  for (let key in target) {
    if (target.hasOwnProperty(key)) {
      ans[key] = target && typeof target === 'object' ? deepClone(target[key]) : target[key]
    }
  }
  return ans
}
```
