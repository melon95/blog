最新的 ECMAScript 标准定义了 8 种数据类型：

- 7种基本类型数据
   - String
   - Number
   - BigInt
   - Null
   - Undefined
   - Symbol
   - Boolean
- 引用类型数据
   - Object

## 数据类型检测
<a name="DQ4wp"></a>
#### typeof
`typeof` 可以用来检测除了Null之外的其他基本数据类型<br />

```javascript
typeof 1 // 'number'
typeof 2n // 'bigint'
typeof '' // 'string'
typeof true // 'boolean'
typeof undefined // 'undefined'
typeof Symbol() // 'symbol'

typeof null // 'object'
typeof function(){} // 'function'
typeof [] // 'object'
typeof {} // 'object'

```


<a name="vTyeP"></a>
#### instanceof
**`instanceof`** 用于检测构造函数的 `prototype` 属性是否出现在某个实例对象的原型链上。<br />

```javascript

[] instanceof Array // true
[] instanceof Object // true

new Object() instanceof Object // true
// 加()是因为{}会被解析成代码块
({}) instanceof Object // true

// 这些构造函数都是函数，函数是特殊的对象
Number instanceof Function // true
String instanceof Function // true
Array instanceof Function // true
Object instanceof Function // true
Function instanceof Object // true
Object instanceof Object // true



// 基本类型
'' instanceof String // false
// new String()是返回一个String对象
new String() instanceof String // true

1 instanceof Number // false
// new Number()是返回一个Number对象
new Number() instanceof Number // true

true instanceof Boolean // false
new Boolean() instanceof Boolean // false

// 下面两种都不能使用new操作符
BigInt(2) instanceof BigInt // false
Symbol('') instanceof Symbol // false

```

<br />手动实现简单 `instanceof` ：<br />

```javascript
  function Instanceof (leftValue, rightValue) {
    let prototype = leftValue.__proto__
    while (prototype) {
      if (prototype === rightValue.prototype) {
        return true
      } else {
        prototype = prototype.__proto__
      }
    }
    return false
  }
```

## 数据类型转换


<a name="AMFqi"></a>
#### 转换成 `Boolean`

- 空字符串、 `null` 、 `undefined` 、0、 `NaN` 都会转换成**false**
- 其他所有值都会转换成true
```javascript
Boolean('0') // true
Boolean(' ') // true
Boolean([])  // true
Boolean({})  // true
```
 
<a name="KSRJc"></a>
#### 转换成 `String` 

- `Number` 、 `null` 、 `undefined` 、 `Boolean` 
```javascript
String(2) // '2'
String(null) // 'null'
String(undefined) // 'undefined'
String(true) // 'true'
String(false) // 'false'
```

- `Object` ：对象尝试把自身转换原始值(primitive)，优先调用 `Symbol.toPrimitive` 方法，其次是 `toString` 方法，最后是 `valueOf` 方法，**如果这三个方法都没有或者不是函数或者返回值都不是基本类型数据**，则会抛出 `TypeError` 
```javascript
Object.prototype.toString = null
let obj = {
  valueOf() {
    return 3
  },
  toString() {
    return 4
  },
  [Symbol.toPrimitive]() {
    return 5
  }
}
let obj1 = {
  valueOf() {
    return 3
  },
  toString() {
    return 4
  },
}

let obj2 = {
  valueOf() {
    return 3
  },
}
let obj3 = { }
console.log(String(obj)) // '5'
console.log(String(obj1)) // '4'
console.log(String(obj2)) // '3'
console.log(String(obj3)) // Uncaught TypeError: Cannot convert object to primitive value
```

- `Array` ：规则跟 `Object` 一样，只不过是重写了 `toString` 方法：对数组的每一项都依次转换成字符串用逗号拼接起来
```javascript
// obj为上面定义的对象
let arr = [obj, 2]
console.log(String(arr)) // '5,2'
arr.toString = function () {
  return '自定义'
}
console.log(String(arr))  // 自定义
arr.toString = null
console.log(String(arr))  // Uncaught TypeError: Cannot convert object to primitive value
```
<a name="H1SBL"></a>
#### 转换成 `Number` 

- `null` 、 `undefined` 、 `Boolean` 
```javascript
Number(null) // 0
Number(undefined) // NaN
Number(true) // 1
Number(false) // 0
```

- `String` ：去掉首尾空格后的纯数字字符串中含有的数字。如果剩余字符串为空，则转换结果为 `0`。否则，将会从剩余字符串中“读取”数字。当类型转换出现 error 时返回 `NaN`。
```javascript
Number('') // 0
Number('2') // 2
Number('2.2') // 2.2
Number('2a') // NaN
Number(' 2 ') // 2
Number('a') // NaN
```

- `Object` ：对象（包括数组）会首先被转换为相应的基本类型值，如果返回的是非数字的基本类型值，则再遵循以上规则将其强制转换为数字。<br /> 转换为基本类型数据的方法还是`Symbol.toPrimitive` 、`valueOf`、`toString` (优先级由高到低，跟转换为字符串有点区别)，**如果这三个方法都没有或者不是函数或者返回值不是基本类型数据**，抛出 `TypeError` 
```javascript
let obj = {
  toString() {
  	return 3
  },
  valueOf() {
  	return 4
  },
  [Symbol.toPrimitive]() {
    return 5
  }
}
let obj1 = {
  toString() {
  	return 3
  },
  valueOf() {
  	return 4
  },
}
let obj2 = {
  toString() {
  	return 3
  },
}
let obj3 = {}
console.log(Number(obj)) // 5
console.log(Number(obj1)) // 4
console.log(Number(obj2)) // 3
console.log(Number(obj3)) // NaN
```

- `Array` ：跟 `Object` 是一样的情况。下面是对 `Array` 的一个总结：
   - 如果数组的长度为0，则转换结果为0。空数组转换成基本数据类型就是 `''` ，在转换成 `Number` 类型就是0。
   - 如果数组的长度为1，则转换结果就相当于直接对数组的第一项直接调用 `Number` 函数
   - 如果数组的长度大于1，则转换结果为NaN。数组长度大于1，转换成基本数据类型就是有逗号拼接的字符串，转换成 `Number` 就是 `NaN` 。

## 相等比较
`JavaScript` 中相等比较分为两种：

1. `==` ：允许在相等比较过程中进行强制类型转换
1. `===` ：不允许在相等比较过程中转换类型

`===` 完全像我们预期的那样执行，但是 `==` 有可能会出乎于我们的意料之外。<br />下面分析一下 `==` 的一些常用情况：

- 字符串和数字转换：会尝试把字符串转换为数字在进行比较
- 其他类型和 `Boolean` 转换：先把 `Boolean` 转换为 `Number` 在进行比较操作
- 引用类型数据和基本属性数据比较：把引用类型转换为原始值(Primitive)，再进行比较
