HTTP结构

一个 HTTP  请求分为两个步骤：发起请求和接受响应。
发起请求(接受响应)的格式分为三个：

  • 首行：请求方式 请求地址 HTTP版本(响应码 响应消息 HTTP版本)
  • 头部信息：分为请求头部信息和响应头部信息，其中请求头和响应头都有的字段则称之为通用头部
  • 空行
  • 主体消息：请求主体消息(响应主体消息)
